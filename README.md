# airplane
Python 飞机大战 V 0.3

## 运行

1. 前往 https://www.python.org/ 下载安装 Python
2. ```pip install pygame```
3. 前往 https://github.com/blackyau/airplane/releases 下载你想要版本的源码
4. 解压后在项目根目录运行 ```python main.py``` 即可

## 实现功能

 - 显示主窗口
 - 响应键盘事件
 - 响应窗口关闭事件
 - 显示敌机
 - 显示自己
 - 使敌机和自己可移动
 - 空格自己发射子弹
 - 敌机自动发射子弹
